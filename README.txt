Ejercicios de Haskell para la asignatura Programacion Declarativa. Éste es el temario de dicha asignatura, y por tanto los conceptos de Haskell que se aplican:

	* Funciones de orden superior y polimorfismo
	* Definición de tipos
	* El sistema de clases de Haskell
	* Programación con listas
	* Programación con árboles
